# Deep Focus Studio 🎧

A neuroscience-backed ambient sound generator for deep work sessions. No dependencies, no build step — just open and focus.

![Deep Focus Studio](https://img.shields.io/badge/Focus-5%2B%20Hours-6366f1)
![License](https://img.shields.io/badge/License-MIT-green)

## Features

### 🧠 Binaural Beats
Scientifically-studied brainwave entrainment:
- **Theta (6 Hz)** — Creative flow, meditation
- **Alpha (10 Hz)** — Relaxed focus, calm alertness  
- **Beta (18 Hz)** — Active thinking, concentration
- **Gamma (40 Hz)** — Peak cognition, memory consolidation

*Requires headphones for the binaural effect to work.*

### 🌊 Noise Textures
- **Brown noise** — Deep, rumbling (like a waterfall)
- **Pink noise** — Balanced (like steady rain)
- **White noise** — Bright, full spectrum

### 🎵 Procedural Chill EDM
Algorithmically generated music that never repeats:
- Evolving arpeggios with ping-pong delay
- Lush pad chords with reverb
- Optional soft beats
- 3 presets: Ambient, Lo-fi, Deep House

### ⏱️ Focus Timer
- Tracks session duration
- Saves stats to localStorage (sessions, total time, streak)
- Persists across browser refreshes

## Usage

### Option 1: GitHub Pages
Visit the live site: `https://YOUR_USERNAME.github.io/deep-focus-studio`

### Option 2: Local
1. Download `index.html`
2. Open in any modern browser
3. Put on headphones
4. Click "Start session"

## The Science

| Layer | Mechanism | Research |
|-------|-----------|----------|
| Binaural beats | Frequency-following response in auditory cortex | Oster (1973), Wahbeh et al. (2007) |
| Pink/Brown noise | Stochastic resonance, distraction masking | Zhou et al. (2012) |
| 40 Hz Gamma | Enhanced neural synchronization | MIT Tsai Lab (2016) |
| Ambient music | Reduced cortisol, improved mood | Thoma et al. (2013) |

## Customization Tips

- **Deep reading/writing**: Alpha + Brown noise
- **Creative brainstorming**: Theta + Ambient EDM
- **Analytical work**: Beta + minimal noise
- **Peak performance sprints**: Gamma (use sparingly)

## Tech Stack

- Vanilla HTML/CSS/JS
- [Tone.js](https://tonejs.github.io/) for Web Audio synthesis
- localStorage for persistence
- Zero build tools, zero dependencies to install
